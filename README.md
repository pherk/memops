# New MemOps for eXist-db


Instructions and documentation on its use is given in the app itself.

## Building

You should have a npm installation at hand.

After cloning run:
```
npm install
```

To build the app:
```
gulp xar
```

To install the xar in eXist-db:
```
gulp install
```
